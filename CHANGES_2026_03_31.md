## 📁 Изменённые файлы

### main.py

**Добавлено:**

1. **Функция `is_catalog_command()`:**
   ```python
   def is_catalog_command(text: str) -> bool:
       """Проверка: команда 'Каталог' (в т.ч. с опечаткой 'Коталог')"""
       text_lower = text.lower().strip()
       return text_lower in ['каталог', 'коталог', 'каталог семян', 'коталог семян']
   ```

2. **Обработчик `@dp.message` с regexp:**
   ```python
   @dp.message(F.text.lower().regexp(r'^(каталог|коталог)(\s*семян)?$'))
   async def cmd_catalog_text(message: Message):
       """Обработчик команды 'Каталог' (в т.ч. с опечаткой 'Коталог')"""
       await message.answer(get_catalog_response())
   ```

3. **Проверка в `handle_user_message()`:**
   ```python
   # Проверка: команда "Каталог" (в т.ч. с опечаткой "Коталог")
   if is_catalog_command(user_text):
       logger.info("Команда 'Каталог' — отправляем полный каталог")
       await message.answer(get_catalog_response())
       return
   ```

## 🔧 Технические детали

### Регулярное выражение:
```regex
^(каталог|коталог)(\s*семян)?$
```

**Расшифровка:**
- `^` — начало строки
- `(каталог|коталог)` — точно "каталог" или "коталог"
- `(\s*семян)?` — опционально: пробелы + "семян"
- `$` — конец строки

### Обработка в группах:
- Если группа с режимом `CHAT_MENTION_ONLY` — требуется упоминание бота
- Если личный чат или `CHAT_AGRO_MODE` — работает без упоминания
