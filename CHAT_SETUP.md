# 🛡️ Настройка бота для работы в двух чатах

## 📋 Описание

Бот может работать в **двух режимах**:

### Чат 1: Режим "По упоминанию" (`CHAT_MENTION_ONLY`)
- Бот отвечает **ТОЛЬКО** когда его упоминают (@Agrio_Bot)
- Пример: `@Agrio_Bot когда сажать томаты?`
- Без упоминания — игнорирует сообщения
- **Идеально для:** общих чатов, групп с обсуждениями

### Чат 2: Режим "Агроном" (`CHAT_AGRO_MODE`)
- Бот отвечает **НА ВСЕ** сообщения по агротематике
- Упоминание не требуется
- Пример: `когда сажать томаты?` → бот отвечает
- **Идеально для:** личного чата, семейной группы

---

## 🔧 Настройка

### Шаг 1: Узнайте chat_id чатов

1. **Добавьте бота** [@getmyid_bot](https://t.me/getmyid_bot) в Telegram
2. **Напишите ему** любое сообщение
3. **Получите ответ:**

```
Ваш ID: 123456789
ID чата: -987654321
```

4. **Запомните ID чата** где хотите использовать бота

### Шаг 2: Настройте .env файл

Откройте файл `.env` (или создайте из `.env.example`):

```bash
# ЧАТ 1: Бот отвечает ТОЛЬКО по упоминанию
# Вставьте chat_id чата/группы
CHAT_MENTION_ONLY=-1001234567890

# ЧАТ 2: Бот отвечает НА ВСЕ сообщения (режим агронома)
# Вставьте chat_id личного чата
CHAT_AGRO_MODE=123456789
```

### Шаг 3: Перезапустите бота

```bash
# Остановить бота (Ctrl+C или taskkill)
taskkill /F /T /PID <PID>

# Запустить заново
python main.py
```

## 📊 Примеры настройки

### Пример 1: Группа + личный чат

```bash
# Группа Telegram (бот по упоминанию)
CHAT_MENTION_ONLY=-1001234567890

# Ваш личный чат с ботом (отвечает всегда)
CHAT_AGRO_MODE=123456789
```

### Пример 2: Две группы

```bash
# Группа 1 (только по упоминанию)
CHAT_MENTION_ONLY=-1001111111111

# Группа 2 (агро-режим, отвечает всем)
CHAT_AGRO_MODE=-1002222222222
```

### Пример 3: Только один чат

```bash
# Только режим по упоминанию
CHAT_MENTION_ONLY=-1001234567890
CHAT_AGRO_MODE=0  # Отключено

# ИЛИ только агро-режим
CHAT_MENTION_ONLY=0  # Отключено
CHAT_AGRO_MODE=123456789
```

## 🎯 Как это работает

### Логика бота:

```python
if PRIVATE_CHATS настроены:
    if chat_id не в PRIVATE_CHATS:
        ИГНОРИРОВАТЬ  # Бот молчит
    
    if chat_id == CHAT_MENTION_ONLY:
        if бот упомянут (@bot):
            ОТВЕЧАТЬ
        else:
            ИГНОРИРОВАТЬ
    
    if chat_id == CHAT_AGRO_MODE:
        ОТВЕЧАТЬ на все сообщения
```

### В группах:
- Всегда проверяется упоминание @bot
- Без упоминания — игнорирует

### В личных чатах:
- Зависит от режима (mention/agro)

## 💡 Советы

### 1. Экономия токенов

Настройте `CHAT_MENTION_ONLY` для больших групп — бот не будет тратить токены на каждое сообщение.

### 2. Удобство

Настройте `CHAT_AGRO_MODE` для личного чата — можно спрашивать без упоминаний.

### 3. Безопасность

Бот работает **ТОЛЬКО** в указанных чатах. Все остальные игнорируются.

## 🛠️ Поиск и устранение проблем

### Бот не отвечает ни в одном чате

**Проверьте:**
1. Файл `.env` — правильно ли указаны ID
2. Логи бота — есть ли сообщения "Чат не в списке разрешённых"
3. Перезапустили ли бота после изменений

### Бот отвечает не там где нужно

**Проверьте:**
1. Правильность `CHAT_MENTION_ONLY` и `CHAT_AGRO_MODE`
2. Не перепутаны ли ID местами

### Как узнать текущий chat_id?

**Способ 1:** Через @getmyid_bot

**Способ 2:** Через логи бота:
```
=== ВХОДЯЩЕЕ СООБЩЕНИЕ ===
От: username (user_id=123456789)
Чат: -987654321, тип: private
```
