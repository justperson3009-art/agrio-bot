# 🤖 Настройка Qwen API (бесплатная замена YandexGPT)

## 📋 Почему Qwen?

| Параметр | YandexGPT | Qwen (Alibaba) |
|----------|-----------|----------------|
| **Стоимость** | Платно | Бесплатно (лимиты) |
| **Качество** | Среднее | Высокое |
| **Контекст** | 8K токенов | 32K токенов |
| **Скорость** | Средняя | Высокая |
| **Русский язык** | ✅ | ✅ Отлично |

---

## 🔑 Получение API ключа

### Шаг 1: Регистрация на Alibaba Cloud

1. Откройте https://www.alibabacloud.com/
2. Нажмите **Sign Up** (верхний правый угол)
3. Заполните форму регистрации
4. Подтвердите email и телефон

### Шаг 2: Активация DashScope

1. Войдите в консоль: https://dashscope.console.aliyun.com/
2. Примите условия использования
3. Активируйте сервис (может потребоваться верификация)

### Шаг 3: Создание API ключа

1. Откройте: https://dashscope.console.aliyun.com/apiKey
2. Нажмите **Create New Key**
3. Скопируйте ключ (начинается на `sk-`)

**Важно:** Сохраните ключ в надёжном месте!

## 🎯 Бесплатные лимиты

Qwen предоставляет **бесплатные токены** для новых пользователей:

| Модель | Бесплатно | После исчерпания |
|--------|-----------|------------------|
| qwen-turbo | 1M токенов/мес | $0.002/1K токенов |
| qwen-plus | 500K токенов/мес | $0.004/1K токенов |
| qwen-max | 100K токенов/мес | $0.012/1K токенов |

**Для бота Agrio:**
- 1 запрос ≈ 500-1000 токенов
- 500K токенов ≈ **500-1000 запросов/мес**
- Этого хватит для **15-30 запросов в день**

## ⚙️ Настройка бота

### 1. Обновите .env файл

**На компьютере:**
```
QWEN_API_KEY=sk-ваш_ключ_qwen
QWEN_MODEL=qwen-plus
```

**На сервере:**
```bash
cd /opt/agrio-bot
nano .env
```

Добавьте/обновите строки:
```
QWEN_API_KEY=sk-ваш_ключ_qwen
QWEN_MODEL=qwen-plus
```

Сохраните: `Ctrl+O` → `Enter` → `Ctrl+X`

### 2. Перезапустите бота

**На сервере:**
```bash
systemctl restart agrio-bot
```

### 3. Проверьте логи

```bash
journalctl -u agrio-bot -f
```

Должно быть:
```
Бот авторизован: @Agrio_Bot
Бот запускается...
```

## 💡 Оптимизация расходов

### 1. Кэширование
Бот уже кэширует ответы (1 час). Повторные вопросы бесплатны.

### 2. Лимит длины ответа
Максимум 1200 символов в `.env`:
```
MAX_RESPONSE_LENGTH=1200
```

### 3. Контекст диалога
Только последние 3 сообщения:
```
CONTEXT_SIZE=3
```

### 4. Модерация запросов
Бот не использует ИИ для:
- Запросов о покупке (готовые ответы)
- Поиска семян (база данных)
- Команд (/start, /help, /seeds)

## ⚠️ Если токены закончились

### Вариант 1: Пополнение счёта

1. DashScope Console → Billing
2. Пополните счёт ($5-10 хватит надолго)
3. Бот продолжит работать

### Вариант 2: Переключение на дешёвую модель

В `.env` измените:
```
QWEN_MODEL=qwen-turbo
```

Перезапустите:
```bash
systemctl restart agrio-bot
```

## 🆘 Troubleshooting

### Ошибка "Invalid API key"

**Решение:**
1. Проверьте ключ в `.env`
2. Убедитесь что нет лишних пробелов
3. Перезапустите бота

### Ошибка "Insufficient balance"

**Решение:**
1. Проверьте баланс в DashScope Console
2. Пополните счёт или переключитесь на qwen-turbo

### Ошибка "Rate limit exceeded"

**Решение:**
1. Увеличьте `API_TIMEOUT` в `prompts.py`
2. Добавьте задержку между запросами
