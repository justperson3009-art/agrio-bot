# Agrio AI Bot

Telegram-бот с ИИ-консультантом для магазина семян Agrio.

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка окружения

Скопируйте `.env.example` в `.env` и заполните:

```bash
cp .env.example .env
```

Заполните файл `.env`:
- `BOT_TOKEN` — токен от @BotFather
- `YANDEX_API_KEY` — API ключ Яндекс Cloud
- `YANDEX_FOLDER_ID` — ID облака из Яндекс Cloud
- `ADMIN_ID` — ваш Telegram ID

### 3. Запуск бота

```bash
python main.py
```

## 🤖 Возможности

- ✅ Консультации по агротехнике через YandexGPT
- ✅ Модерация сообщений (мат, оффтоп)
- ✅ Защита от Prompt Injection
- ✅ Контекст диалога (последние 5 сообщений)
- ✅ Логирование всех запросов к ИИ
- ✅ Ограничение длины ответа (500 символов)

## 📝 Логи

Логи запросов к ИИ сохраняются в `logs/ai_requests.log`

## ⚙️ Настройки

В `prompts.py` можно изменить:
- `SYSTEM_PROMPT` — инструкция для ИИ
- `AGRO_KEYWORDS` — ключевые слова агротематики
- `STOP_WORDS` — стоп-слова для модерации
- `MAX_RESPONSE_LENGTH` — лимит символов ответа
- `CONTEXT_SIZE` — размер контекста диалога
