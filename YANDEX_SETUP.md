# 🔑 НАСТРОЙКА YANDEXGPT НА СЕРВЕРЕ

**Дата:** 31 марта 2026

---

## 📋 ШАГИ

### 1. Найти ключи в файле "ключь яндекс.txt"

**На компьютере откройте файл:**
```
C:\Users\user\Desktop\Agrio Bot\ключь яндекс.txt
```

**Там будут 2 значения:**
- Идентификатор ключа (начинается с `aje...`)
- Секретный ключ (начинается с `AQVN...`)

---

### 2. Найти Folder ID в Яндекс.Облаке

1. Откройте https://console.cloud.yandex.ru/
2. Войдите в ваш аккаунт
3. На главной странице выберите каталог
4. Вверху страницы будет **ID каталога** (начинается с `b1g...`)
5. Скопируйте этот ID

---

### 3. Обновить .env на сервере

**На сервере выполните:**

```bash
cd /opt/agrio-bot
nano .env
```

**Добавьте в конец файла (3 строки):**

```env
# YandexGPT API
YANDEX_KEY_ID=ваш_key_id_из_файла
YANDEX_SECRET_KEY=ваш_secret_key_из_файла
YANDEX_FOLDER_ID=ваш_folder_id_из_Яндекс
```

**Сохраните:**
- `Ctrl+O` → `Enter` (сохранить)
- `Ctrl+X` (выйти)

---

### 4. Перезапустить бота

```bash
systemctl restart agrio-bot
systemctl status agrio-bot --no-pager
```

---

### 5. Проверить работу

**В Telegram:**
1. `/start` - бот отвечает ✅
2. `Каталог` - показывает все семена ✅
3. `Когда сажать томаты?` - отвечает из базы ✅

---

## 🆘 Если ошибка

```bash
# Проверка логов
journalctl -u agrio-bot -n 30 --no-pager

# Проверка .env
cat /opt/agrio-bot/.env | grep YANDEX
```
