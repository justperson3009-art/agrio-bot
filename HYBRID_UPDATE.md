## ✅ ПРОВЕРКА РАБОТЫ

### Тест 1: Простой вопрос (база знаний)
```
Когда сажать томаты?
```
**Ожидаемо:** Мгновенный ответ из шаблона (источник: `simple`)

### Тест 2: Вопрос о семенах (база данных)
```
Какие томаты у вас есть?
```
**Ожидаемо:** Список сортов из БД (источник: `seeds_db`)

### Тест 3: Сложный вопрос (YandexGPT)
```
Как правильно формировать томаты в теплице?
```
**Ожидаемо:** Развёрнутый ответ от ИИ (источник: `ai`)

### Тест 4: Покупка (готовый ответ)
```
Где купить семена?
```
**Ожидаемо:** Информация о магазинах (источник: `purchase_info`)

## 📊 ИСТОЧНИКИ ОТВЕТОВ

| Источник | Когда используется | Пример |
|----------|-------------------|--------|
| `simple` | Простые вопросы (контакты, доставка, цены) | "Какой сайт?" |
| `seeds_db` | Вопросы о сортах семян | "Какие томаты есть?" |
| `purchase_info` | Запросы о покупке | "Где купить?" |
| `ai` | Сложные вопросы, консультации | "Как формировать томаты?" |
| `fallback` | Простые вопросы без ответа в базе | "Привет" |

## 🔧 НАСТРОЙКА YANDEXGPT

### Создание сервисного аккаунта

1. **Зайдите в Яндекс.Облако:** https://console.cloud.yandex.ru/

2. **Создайте сервисный аккаунт:**
   - IAM → Сервисные аккаунты → Создать
   - Имя: `agrio-bot`
   - Роль: **`ai.languageModels.user`**

3. **Создайте ключи авторизации:**
   - Вкладка "Ключи" → Создать новый ключ
   - Тип: **Ключ авторизации**
   - Скопируйте **Key ID** (YCAJ...) и **Secret Key** (YCP...)

4. **Получите Folder ID:**
   - Главная консоль → выберите каталог
   - Скопируйте ID (начинается с `b1g...`)

## 🆘 ВОЗМОЖНЫЕ ПРОБЛЕМЫ

### Ошибка: "YANDEX_API_KEY и YANDEX_FOLDER_ID должны быть указаны"

**Решение:** Добавить ключи в `.env` на сервере

### Бот не отвечает

**Проверка:**
```bash
# Статус сервиса
systemctl status agrio-bot

# Логи
journalctl -u agrio-bot -n 50 --no-pager
```

### Ошибка YandexGPT API

**Причины:**
- Неверный API ключ
- Превышен лимит токенов
- Нет роли `ai.languageModels.user`

**Проверка:**
```bash
# Проверка .env
cat /opt/agrio-bot/.env | grep YANDEX
```
