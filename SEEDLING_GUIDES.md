# 📚 БАЗА ЗНАНИЙ: РАССАДА

## ✅ Добавленные шаблоны (март 2026)

### 🍅 Томаты
**Ключевые слова:** рассада томат, томаты рассада, помидоры рассада, рассада помидор

**Что включает:**
- 📅 Сроки посева (ранние/средние)
- 🌡️ Температурный режим (3 этапа)
- 💡 Освещение (14-16 часов)
- 💧 Полив (до/после всходов)
- 🌿 Пикировка (2 настоящих листа)
- 🥄 Подкормка (2 этапа)
- ⚠️ Проблемы и решения

---

### 🌶️ Перец
**Ключевые слова:** рассада перц, перец рассада, рассада болгарского перца

**Особенности:**
- ❌ НЕ рекомендуется пикировка
- 💧 Только тёплая вода (+25°C)
- 🌡️ Высокая температура до всходов (+28-30°C)

---

### 🍆 Баклажаны
**Ключевые слова:** рассада баклажан, баклажаны рассада, синенькие рассада

**Особенности:**
- ❌ НЕ рекомендуется пикировка
- 💧 Обильный, но редкий полив
- 🌡️ Теплолюбивая культура

---

### 🥒 Огурцы
**Ключевые слова:** рассада огурц, огурцы рассада, огурцы выращивание

**Особенности:**
- ⛔ КАТЕГОРИЧЕСКИ нельзя пикировать
- 📅 Короткий срок рассады (25-30 дней)
- 💧 Почва всегда влажная

---

### 🥬 Капуста
**Ключевые слова:** рассада капуст, капуста рассада, капуста выращивание

**Особенности:**
- 🌡️ Любит прохладу (+15-18°C днём)
-  Разные сроки для ранней/средней/поздней
- ⚠️ Риск чёрной ножки

---

### 🥒 Кабачки
**Ключевые слова:** рассада кабачк, кабачки рассада, кабачки выращивание

**Особенности:**
- ❌ НЕ пикировать
- 📅 Короткий срок (25-30 дней)
- 🥄 Подкормка органикой + зола

---

### 🎃 Тыква
**Ключевые слова:** рассада тыкв, тыква рассада, тыква выращивание

**Особенности:**
- ❌ НЕ пикировать
- 💧 Умеренный полив
- 🥄 Подкормка золой

---

### 🌿 Зелень
**Ключевые слова:** рассада зелени, петрушка рассада, укроп рассада, зелень выращивание

**Особенности:**
- 🌡️ Любит прохладу (+15-18°C)
- 📅 Петрушка: 30-35 дней
- 🌿 Укроп: можно сразу в грунт

## ➕ Как добавить новый шаблон

1. **Откройте** `hybrid_ai.py`

2. **Найдите** раздел `# === РАССАДА: ... ===`

3. **Добавьте** новый шаблон по аналогии:

```python
# === РАССАДА: [КУЛЬТУРА] ===
'[culture]_seedling': {
    'patterns': ['рассада [культура]', '[культура] рассада'],
    'answer': (
        "🔤 **[НАЗВАНИЕ]**\n\n"
        "📅 **Сроки:** ...\n"
        "🌡️ **Температура:** ...\n"
        # и т.д.
    )
},
```

4. **Сохраните** и проверьте:
```bash
python -m py_compile hybrid_ai.py
```
