## 🆕 Новые команды бота

### `/seeds` - Показать каталог семян

Выводит краткую сводку по всем категориям семян с количеством сортов.

**Пример:**
```
/seeds
```

**Ответ:**
```
🌱 КАТАЛОГ СЕМЯН AGRIO.BY

Всего в каталоге: 104 сортов и гибридов

🍉 Арбуз — 7 шт.
🍆 Баклажан — 3 шт.
🥦 Брокколи — 7 шт.
...
```

### `/seed [запрос]` - Поиск семян

Поиск по названию, категории или характеристикам.

**Примеры использования:**

```
/seed томат           # Все томаты
/seed СУПЕРНОВА       # Конкретный сорт
/seed ранний томат    # Поиск по характеристикам
/seed теплица         # Поиск по условиям выращивания
```

Бот автоматически распознаёт запросы о покупке и показывает информацию без агротехнических рекомендаций.

### Ключевые слова для распознавания:

- `купить`, `где найти`, `где приобрести`, `приобрести`, `взять`

## 🔍 Критерии поиска

Бот распознаёт следующие критерии:

### Сроки созревания:
- `ранний` / `раннеспелый`
- `среднеранний`
- `средний`
- `среднепоздний`
- `поздний`
- `ультраранний`

### Условия выращивания:
- `теплица` / `теплицы`
- `открытый грунт` / `ОГ`

### Назначение:
- `хранение` / `для хранения`
- `переработка` / `для переработки`
- `рынок` / `для продажи`
- `свежее потребление`

### Особенности:
- `крупный` / `крупноплодный`
- `сладкий`
- `урожайный`

## 💡 Интеграция с ИИ

База семян интегрирована в систему ИИ-консультанта:

1. **При запросе о семенах** — бот сначала ищет рекомендации в базе
2. **При формировании ответа** — ИИ получает список рекомендуемых сортов
3. **В ответе** — ИИ рекомендует конкретные сорта из ассортимента AGRIO

### Пример полного ответа:

```
🌶️ ПЕРЕЦ СЛАДКИЙ — ПОЛНАЯ ИНСТРУКЦИЯ

📅 СРОКИ:
• Ранние сорта: 15-20 февраля
• Средние: 1-10 марта
• Поздние: 1-5 февраля

🌱 ПОДГОТОВКА:
...

🏪 РЕКОМЕНДУЕМЫЕ СОРТА AGRIO:
• ГЕРКУЛЕС F1 — кубовидный, 220-250г, стенка 7-10мм
• РЕД ДЖЕВЕЛ F1 — красный премиум
• ВАНГАРД F1 — ранний, для теплиц и ОГ
```

## 🛠️ API базы данных

### Основные методы:

```python
from seeds_database import seeds_db

# Получить все категории
categories = seeds_db.get_all_categories()

# Получить семена по категории
tomatoes = seeds_db.get_seeds_by_category('томат')

# Найти по названию
seed = seeds_db.get_seed_by_name('СУПЕРНОВА F1')

# Поиск по названию (частичное совпадение)
results = seeds_db.search_by_name('СУПЕР')

# Поиск по характеристикам
results = seeds_db.search_by_features(['ранний', 'теплица'])

# Получить рекомендации с критериями
recommendations = seeds_db.get_recommendations_for_category(
    'томат',
    {'ripening_period': 'ранн', 'growing_conditions': 'теплиц'}
)

# Форматировать информацию
info = seeds_db.format_seed_info(seed)
```
