# ✅ Оптимизация промпта и ответов ИИ

## 📊 Что изменено

### 1. Сокращение системного промпта

| Параметр | До | После | Экономия |
|----------|-----|-------|----------|
| Длина промпта | ~4500 символов | **~2600 символов** | **42%** |
| Токены на запрос | ~1500 | **~900** | **40%** |

**Убрано:**
- ❌ Дублирование описаний сортов
- ❌ Развёрнутые примеры (1000+ символов)
- ❌ Многословные требования
- ❌ Повторы "ОБЯЗАТЕЛЬНО", "ЗАПРЕЩЕНО"

**Осталось:**
- ✅ Весь ассортимент (104 сорта)
- ✅ Структура ответа
- ✅ Ключевые требования
- ✅ Краткий пример

### 2. Уменьшение длины ответа

| Параметр | До | После | Экономия |
|----------|-----|-------|----------|
| MAX_RESPONSE_LENGTH | 2000 символов | **1200 символов** | **40%** |
| CONTEXT_SIZE | 5 сообщений | **3 сообщения** | **40%** |

### 3. Новые требования к ответу

**Было:**
```
1. Минимум 400-600 символов — подробные развёрнутые ответы
2. СТРУКТУРА обязательна: 📅Сроки → 🌱Подготовка → 🌿Посадка → 💧Уход → 🐛Защита
3. Конкретные цифры: даты, температуры (°C), расстояния (см), сроки (дни)
4. Эмодзи для каждого раздела
5. Для ПЕРЦЕВ и ТОМАТОВ — отдельные инструкции по сортам
6. При рекомендации семян — указывай КОНКРЕТНЫЕ сорта
```

**Стало:**
```
1. 300-500 символов — кратко, содержательно, без воды
2. Структура: 📅Сроки → 🌱Подготовка → 🌿Посадка → 💧Уход → 🐛Защита
3. Конкретика: даты, °C, см, дни
4. Эмодзи в заголовках
5. 2-3 сорта AGRIO в конце
```

### 4. Пример ответа

**Было (многословно):**
```
🌶️ ПЕРЕЦ СЛАДКИЙ — ПОЛНАЯ ИНСТРУКЦИЯ

📅 СРОКИ:
• Ранние сорта: 15-20 февраля
• Средние: 1-10 марта
• Поздние: 1-5 февраля
• Высадка: через 60-70 дней

🌱 ПОДГОТОВКА:
1. Эпин (2 капли/100мл) — 12 часов
2. Проращивание: влажная ткань, +25-28°C, 3-5 дней
3. Посев: сразу после наклевывания
...
```

**Стало (компактно):**
```
🌶️ ПЕРЕЦ — ИНСТРУКЦИЯ

📅 СРОКИ: Ранние 15-20 фев, средние 1-10 мар, высадка через 60-70 дней

🌱 ПОДГОТОВКА: Эпин 12ч → влажная ткань +25-28°C 3-5 дней → посев

🌿 ПОСАДКА: 0.5-1см глубина, стаканчики 0.5л (перец не любит пересадку!)

💧 УХОД: +22-25°C днём, свет 12-14ч, полив тёплой водой, подкормка через 14 дней

🐛 ЗАЩИТА: Без сквозняков, не заглублять корневую шейку

🏪 СОРТА AGRIO: **ГЕРКУЛЕС F1** (кубовидный, стенка 7-10мм), **ВАНГАРД F1** (ранний), **РЕД ДЖЕВЕЛ F1** (красный премиум)
```

## 💰 Экономия токенов

### На один запрос:

| Компонент | До | После | Экономия |
|-----------|-----|-------|----------|
| Системный промпт | ~1500 токенов | ~900 токенов | 600 токенов |
